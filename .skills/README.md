# Antigravity Skills

To add custom skills or complex workflows for me (Antigravity) to follow, you should:

1. Create a `.skills/` directory in your project root.
2. Add `.md` or `.skill` files there with detailed instructions.
3. When you want me to perform a complex task, you can tell me to "Use the X skill".

Example Skill File Structure:
- `deploy.skill`: Instructions for production deployment.
- `database-migration.skill`: Steps for safely updating the schema.
- `ui-guidelines.skill`: Visual and branding rules to follow.

I will automatically look for these files when you mention them.
